Quick Start

How to Embed

Add this single line of code anywhere in your HTML. The widget will render automatically.

HTML <script src="https://growvpd.pro/widget.js"></script>

Paste this into your page's HTML where you want the calculator to appear. No registration, no API key, no dependencies.

Customize

Configuration Options

Use data attributes to customize the widget's behavior and appearance.

data-theme="dark"

Switch the widget to a dark color scheme. Ideal for forums and sites with dark backgrounds.

data-target="vpd"
