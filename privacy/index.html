Privacy Policy

Last updated: April 6, 2026

Overview

GrowVPD Pro ("we", "our", "the app") is committed to protecting your privacy. This app is designed with an offline-first architecture. We do not collect, store, or transmit any personal data.

Data Storage

All data you enter in GrowVPD Pro is stored exclusively on your device:

  • Grow diary entries, photos, and notes
  • VPD calculations and settings
  • Equipment profiles and templates
  • App preferences (language, theme, units)

No data is uploaded to any server. When you uninstall the app, all data is permanently deleted from your device.

Smart Device Connections (Pro Feature)

If you choose to connect smart devices, the app communicates with the following third-party services using credentials you provide (email/password or API keys):

  • Tuya Cloud API — device control and status
  • AC Infinity API — device control and status (note: uses HTTP, not HTTPS)
  • Mars Hydro API — device control and status
  • Spider Farmer API — device control and status
  • Vivosun API — device control and status
  • Bluelab Edenic API — pH/EC/temperature sensor data

Your credentials for these services are:

  • Transmitted only to the respective platform's API for authentication and device control
  • Never sent to our servers (we have no servers)
  • Stored securely on your device using Android Keystore encryption or iOS Keychain
  • Deleted when you disconnect the platform or uninstall the app

Sensor Data

The app collects environmental sensor data from your connected devices, including temperature, humidity, VPD, CO2, PPFD, and soil metrics. All sensor data is stored locally on your device and is not sent to any external server.

Camera and Photos

The app may request camera and storage permissions to:

  • Take photos for your grow diary
  • Capture smart camera snapshots (Tuya cameras)
  • Browse the disease & pest identification guide

All photos and media are stored locally on your device. They are never uploaded to any external server. The optional "Photo Privacy" feature can hide grow photos from your device's gallery.

Bluetooth and Location

The app may request Bluetooth LE permissions to connect to temperature/humidity sensors and smart devices (e.g., Spider Farmer, Mars Hydro). All Bluetooth data is processed locally and not transmitted to any server.

On Android, the app requests location permission solely because it is required by Android for BLE scanning. The app does not track your location or use location data for any other purpose.

Advertising and Analytics

GrowVPD Pro does not include any advertising SDKs or analytics frameworks. We do not use Google AdMob, Firebase Analytics, or any other tracking tools. No usage data is collected or transmitted.

Third-Party Services

The app communicates with the following third-party services only when you explicitly connect to them:

  • Google Play Billing — Pro purchases (Android)
  • Tuya Cloud API — smart device control (Pro feature, user-initiated)
  • AC Infinity API — smart device control (Pro feature, user-initiated)
  • Mars Hydro API — smart device control (Pro feature, user-initiated)
  • Spider Farmer API — smart device control (Pro feature, user-initiated)
  • Vivosun API — smart device control (Pro feature, user-initiated)
  • Bluelab Edenic API — sensor data (Pro feature, user-initiated)

No data is sent to any service unless you explicitly connect your account. We do not operate any servers of our own.

Children's Privacy

GrowVPD Pro is not intended for children under 18. We do not knowingly collect data from children.

Data Deletion

Since all data is stored locally on your device, you can delete it at any time by:

  • Clearing app data in your device settings
  • Uninstalling the app

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated date.
