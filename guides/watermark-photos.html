Photo Watermarking (Pro)

What Is Photo Watermarking?

Every diary photo you take in GrowVPD Pro can be auto-stamped with the data that surrounds it: strain name, day of grow, current stage, today’s VPD reading, harvest week. Stamping happens on save, in the corner of your choice, with a translucent style that does not distract from the plant.

Use cases:

  • Discord/forum sharing — one image carries all context, no caption needed.
  • Cycle comparison — flip through a year of photos and instantly see strain + day on each.
  • Grow journals — PDF exports become polished cycle reports.

Watermarking is a Pro feature.

Enabling Watermark

  1. Settings → Watermark.
  2. Toggle Enable photo watermarking.
  3. Pick a position: bottom-left, bottom-right, top-left, top-right, or center.
  4. Pick a style: Pill (pill-shaped tag), Bar (full-width bottom strip), or Corner (small box).
  5. Pick which fields to include: any combination of strain, day, stage, VPD, week, harvest stats.

From the next diary photo onward, every image carries the watermark. Existing photos are not retroactively stamped — the original file on disk is unchanged.

Per-Grow Override

Each grow has its own watermark toggle in Grow Settings → Photo watermark. Useful if you want stealth photos for one tent (turn off) while marking the rest. Off-grow setting overrides the global default.

Watermark Anatomy

A typical bottom-right pill watermark looks like:

Wedding Cake · Day 42 · Flower Wk 3 · VPD 1.18

Fields are auto-populated from:

  • Strain — from the grow’s strain field.
  • Day — days since grow start.
  • Stage — from your current stage marker (Veg / Flower Wk N).
  • VPD — from the latest sensor reading at photo time, or from the entry you saved with the photo.
  • Harvest stats — only after harvest: dry weight, yield/W.

Style Customisation

Pro users can fine-tune:

  • Background opacity (0–100%).
  • Text colour (white / cream / accent).
  • Font size (small / medium / large — auto-scales to image dimensions).
  • Show/hide the GrowVPD logo.

Preview updates live in the settings screen.

Privacy & Stealth Mode

If you have the “Photo Privacy” or stealth-mode features enabled, watermarking respects them:

  • Stealth mode → watermark omits strain name (replaces with grow ID hash).
  • Stamp uses a generic plant glyph instead of the GrowVPD logo when stealth is on.

Sharing

Tap any photo → Share. The shared file already has the watermark baked in (it’s saved to disk that way). PDF diary exports also include the stamped photos. JSON exports include both stamped JPEGs and the raw originals.

Reverting. The original photo file is preserved untouched in the app’s photo storage. Watermark is rendered into a derivative file only. If you change watermark settings, future photos use the new style; existing ones keep their original stamp.
